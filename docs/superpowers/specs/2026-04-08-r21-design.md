# Round 21

## Feature 1: Section Locking
Lock specific headings/sections so only certain users can edit them.
- New Yjs map: `ydoc.getMap("sectionLocks")` → { headingId: { lockedBy, lockedAt } }
- Right-click heading → "Lock this section"
- Locked sections show a lock icon and are read-only for other users
- Owner and locker can unlock

## Feature 2: AI Auto-Translate
Translate entire document to another language via AI.
- New API: POST /api/agent/translate — sends markdown + target language to Anthropic
- Returns translated markdown
- Button in TopBar: "Translate" dropdown with language options (Spanish, French, Japanese, Chinese, German, Portuguese)
- Creates a new document with translated content (preserves original)

## Feature 3: Breadcrumb Block
Inline navigation showing document hierarchy.
- Custom node that renders folder path of the current doc
- Auto-populated from folder metadata
- Slash command: `/breadcrumb`
- Renders as: Home > Folder > Subfolder (clickable links)

## Feature 4: Zen Mode
Enhanced focus mode with visual dimming.
- Beyond existing focus mode: also dims editor edges, adds vignette effect
- Hides word count, health score, focus timer — shows ONLY text
- Larger text (1.2x), more line spacing
- Enter with Cmd+Shift+Z or "Zen" button
- Click anywhere outside text area to exit

## Feature 5: Duplicate with Options
Enhanced document duplication dialog.
- When clicking "Duplicate", show options dialog:
  - Include comments? (checkbox)
  - Include version history? (checkbox)
  - Include tags? (checkbox)
  - New title: (input, defaults to "Copy of {title}")
- API: POST /api/documents/[id]/duplicate with options

## Feature 6: Quick Reactions on Documents
Emoji reactions on document list rows (like Slack reactions on messages).
- Small reaction bar on each document row
- Click "+" to add reaction from preset emojis
- Store in Prisma: DocumentReaction (id, documentId, userId, emoji, createdAt)
- Show grouped reaction counts

## Feature 7: Editor Minimap
VS Code-style minimap scrollbar on the right edge.
- Render a scaled-down version of the document content
- Highlight visible viewport area
- Click/drag to scroll
- Toggle on/off in settings
