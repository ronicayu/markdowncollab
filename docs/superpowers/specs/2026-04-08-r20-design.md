# Round 20

## Feature 1: Command-K Universal Search
Unified search that finds documents, slash commands, and settings.
- Enhance existing CommandPalette (Cmd+P) to also search: slash commands, toolbar actions, settings
- Tabs at top: "Documents" | "Commands" | "Settings"
- Commands tab: lists all slash commands + toolbar actions, executing them directly
- Settings tab: links to theme, language, toolbar customization

## Feature 2: Outline Numbering in Sidebar
Show numbered headings (1., 1.1., 1.2.) in the outline sidebar.
- Read OutlineSidebar.tsx
- Add numbering based on heading hierarchy (same logic as auto-numbering CSS but computed)
- Toggle on/off (follows the auto-numbering toggle state)

## Feature 3: Cursor Chat (Figma-style)
Ephemeral inline messages at cursor position for quick communication.
- Press `/` then type a message that appears as a floating bubble near your cursor
- Other users see it via Yjs awareness: `awareness.setLocalStateField("chat", {message, timestamp})`
- Messages auto-dismiss after 5 seconds
- Rendered as colored bubbles near each user's cursor

## Feature 4: DOCX Export
Export documents as Microsoft Word format.
- Install: `npm install docx` (officegen alternative)
- New API: GET /api/documents/[id]/export/docx
- Convert Yjs content to DOCX using the `docx` package
- Add "Word (.docx)" option in export dropdown

## Feature 5: Nested Folder Drag & Drop
Drag documents between folders on the document list page.
- Add HTML5 drag-and-drop on document rows and folder tree items
- Drag a document row → drop on a folder → moves doc to that folder
- Visual indicator: folder highlights on dragover
- API call: PUT /api/documents/[id] with new folderId

## Feature 6: Session History Panel
Show last 10 actions with ability to undo any of them.
- Track actions in component state: [{type, description, timestamp, undoFn}]
- Small collapsible panel in the editor (clock icon in footer)
- Click any past action to undo it
- Actions tracked: formatting changes, block insertions, deletions

## Feature 7: Typewriter Mode
Keep the current editing line vertically centered on screen.
- Toggle in TopBar or footer
- When active: scroll the editor so the cursor line is always at ~40% of viewport height
- Use ProseMirror `scrollIntoView` with custom logic
- Stored in localStorage per document
